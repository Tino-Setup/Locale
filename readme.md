<h3 align="center"><em>Locales files for Tino Setup suite</em></h3>

---

#### The locale files are of two parts 1. Installer's locales 2. Uninstaller's locales :

#### Installer's locales are in /installer directory also there are  5 languages in total English(en_US), French(fr_FR), Spanish(es_ES), Hindi(hi_IN), and Chinese(zh_CN).

#### Uninstaller's locales are in /uninstaller directory also there are  5 languages in total English(en_US), French(fr_FR), Spanish(es_ES), Hindi(hi_IN), and Chinese(zh_CN).

#### Each also having the template files for the new languages.
